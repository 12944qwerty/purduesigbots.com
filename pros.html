PROS

PROS is a lightweight and fast alternative operating system for the VEX Cortex Microcontroller. It features multitasking, low-level control, and Wiring compatible functions to harness the full power of the Cortex. With a real-time kernel based on FreeRTOS (https://www.freertos.org) and a development environment built on Eclipse CDT (https://www.eclipse.org/), PROS uses proven software to increase the reliability and usability of the VEX development system across all three major computing platforms.
Eclipse and FreeRTOS have their own licensing agreements. Please see https://www.eclipse.org/legal/epl-v10.html and https://www.freertos.org/a00114.html for details.
